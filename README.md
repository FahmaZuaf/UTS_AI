# Waste Classification Using ANN and CNN

Proyek ini merupakan implementasi klasifikasi citra sampah menggunakan metode **Artificial Neural Network (ANN)** dan **Convolutional Neural Network (CNN)** dengan bantuan library TensorFlow dan Keras.

## Dataset
Dataset yang digunakan berisi 3 kelas sampah, yaitu:

- Organic
- Recyclable
- Non_Recyclable

Dataset gambar diproses menggunakan preprocessing dan augmentasi sebelum dilakukan training model.

---

# Tujuan Proyek

Tujuan dari proyek ini adalah:

- Membangun model klasifikasi citra sampah
- Membandingkan performa ANN dan CNN
- Mengevaluasi model menggunakan accuracy, confusion matrix, dan classification report
